About This Project
GovWatch

GovWatch is a prototype civic transparency tool that aggregates publicly available data about the United States Congress. All data is sourced from official government APIs and the ProPublica Congress API.

This is an early-stage prototype. Future versions will include Florida state legislature tracking, voting pattern analysis, committee hearing schedules, bill notifications, and more.

Data Sources
  • ProPublica Congress API
  • Congress.gov
  • GovTrack.us
  • US Senate & House official websites
